# Вариант 1
1. В декартовых координатах проекция скорости $v_y$ определяется как:
  a) $\ddot{y}$;
  b) $\dot{y}$;
  c) $\int y, dt$;
  d) $y$.

2. Закон движения в декартовых координатах:
$x(t) = 4t^2,\quad y(t) = 2t,\quad z(t) = 5t$.
Чему равна проекция ускорения $a_y$ при $t = 2$ с?
  a) $8\ \text{м/с}^2$;
  b) $4\ \text{м/с}^2$;
  c) $16\ \text{м/с}^2$;
  d) $32\ \text{м/с}^2$.

3. В цилиндрической системе координат коэффициент Ламе $H_z$ равен:
  a) $0$;
  b) $\rho$;
  c) $1$;
  d) $\varphi$.

4. Частица движется по закону
$\rho(t) = 3t,\quad \varphi(t) = 2t^2$
Найдите модуль скорости $|\vec{v}|$ в момент $t = 1$ с.
  a) $\sqrt{9 + 36 + 0}\ \text{м/с}$;
  b) $\sqrt{9 + 144 + 0}\ \text{м/с}$;
  c) $\sqrt{9 + 72 + 0}\ \text{м/с}$;
  d) $\sqrt{9 + 108 + 0}\ \text{м/с}$.

5. При естественном способе описания нормальное ускорение $\vec{a}_n$ вычисляется по формуле:
  a) $\vec{v}$;
  b) $\frac{v^2}{R} \vec{n}$;
  c) $\vec{s}$;
  d) $\frac{\partial^2}{\partial t^2}$.

6. Частица движется по окружности радиуса $R = 6\ \text{м}$.
Закон движения:
$s(t) = 2t^3 + t.$
Чему равно нормальное ускорение $|\vec{a}_n|$ при $t = 1$ с?
a) $6.0\ \text{м/с}^2$;
b) $8.17\ \text{м/с}^2$;
c) $12.5\ \text{м/с}^2$;
d) $24.0\ \text{м/с}^2$.

# Вариант 4
1. Коэффициенты Ламе в декартовой системе координат равны:
a) $x,\ y,\ z$;
b) $\dot{x},\ \dot{y},\ \dot{z}$;
c) $1,\ 1,\ 1$;
d) $0,\ 0,\ 0$.

2. Дано: $a_x(t) = 4t,\ v_x(0) = 0,\ x(0) = 0$. Найдите $x(t)$.
a) $\frac{2t^3}{3}$;
b) $2t^2$;
c) $4t$;
d) $0$.

3. В сферической системе координат локальный базисный вектор $\vec{e}_r$ равен:
a) $(\sin \theta \cos \varphi,\ \sin \theta \sin \varphi,\ \cos \theta)$;
b) $(\cos \theta \cos \varphi,\ \cos \theta \sin \varphi,\ -\sin \theta)$;
c) $(-\sin \varphi,\ \cos \varphi,\ 0)$;
d) $(\sin \theta \cos \varphi,\ \sin \theta \cos \varphi,\ \cos \theta)$.

4. Частица движется в сферической системе координат:
$r(t) = 4,\quad \theta(t) = \pi t,\quad \varphi(t) = t.$
При $t = 1.5$ с $\sin(\pi/2) = 1$. Найдите модуль скорости $|v|$:
a) $\sqrt{0 + 16\pi^2 + 16}\ \text{м/с}$;
b) $\sqrt{0 + 4\pi^2 + 4}\ \text{м/с}$;
c) $\sqrt{0 + 16\pi^2 + 4}\ \text{м/с}$;
d) $\sqrt{0 + 64\pi^2 + 16}\ \text{м/с}$.

5. Кривизна траектории $k$ определяется как:
a) $\left|\frac{d\vec{r}}{ds}\right|$;
b) $\left|\frac{d\vec{\tau}}{ds}\right|$;
c) $\left|\frac{ds}{dt}\right|$;
d) $\left|\frac{d\vec{v}}{dt}\right|$.

6. Частица движется по кривой: $v(t) = 5t,\ R = 2$ м. Найдите нормальное ускорение $|a_n|$ при $t = 2$ с.
a) $50\ \text{м/с}^2$;
b) $25\ \text{м/с}^2$;
c) $12.5\ \text{м/с}^2$;
d) $100\ \text{м/с}^2$.

# Вариант 5
1. В цилиндрической системе координат квадрат скорости $|v|^2$ равен:
a) $\dot{\rho}^2 + \rho^2 \dot{\varphi}^2$;
b) $\dot{\rho}^2 + \rho^2 \dot{\varphi}^2 + \dot{z}^2$;
c) $\dot{\rho}^2 + \rho^2 \dot{\varphi}^2 + \dot{z}^2 \cdot z^2$;
d) $\dot{\rho}^2 + \rho^2 \dot{\varphi}^2 + \dot{z}^2$.

2. Закон движения:
$x(t) = 3t^2,\quad y(t) = 2t^3,\quad z(t) = 1.$
Чему равна проекция $v_y$ при $t = 1$ с?
a) $2$;
b) $4$;
c) $6$;
d) $8$.

3. Вектор скорости $\vec{v}$ в локальном базисе записывается как:
a) $\vec{v} = H_1 \dot{q}_1 \vec{e}_1 + H_2 \dot{q}_2 \vec{e}_2 + H_3 \dot{q}_3 \vec{e}_3$;
b) $\vec{v} = \dot{q}_1 \vec{e}_1 + \dot{q}_2 \vec{e}_2 + \dot{q}_3 \vec{e}_3$;
c) $\vec{v} = H_1 \dot{q}_1 \vec{e}_1 + H_2 \dot{q}_2 \vec{e}_2 + H_3 \dot{q}_3 \vec{e}_3 \cdot q_3$;
d) $\vec{v} = \dot{q}_1 \vec{e}_1 + \dot{q}_2 \vec{e}_2 + \dot{q}_3 \vec{e}_3 \cdot q_3$.

4. Частица движется по закону:
$\rho(t) = 2t^2,\quad \varphi(t) = t,\quad z(t) = 3t^2.$
Найдите модуль скорости $|v|$ при $t = 1$ с:
a) $\sqrt{16 + 36}\ \text{м/с}$;
b) $\sqrt{16 + 4 + 36}\ \text{м/с}$;
c) $\sqrt{16 + 8 + 36}\ \text{м/с}$;
d) $\sqrt{8 + 4 + 18}\ \text{м/с}$.

5. Вектор нормали $\vec{n}$ при естественном способе описания направлен:
a) по касательной к траектории;
b) к центру кривизны;
c) по радиус-вектору;
d) против скорости.

6. Частица движется по окружности $R = 3$ м, $s(t) = t^3 + 2t$. Найдите нормальное ускорение $|a_n|$ при $t = 2$ с:
a) $12\ \text{м/с}^2$;
b) $48\ \text{м/с}^2$;
c) $27\ \text{м/с}^2$;
d) $75\ \text{м/с}^2$.

# Вариант 6
1. В сферической системе координат квадрат скорости $|v|^2$ равен:
a) $\dot{r}^2 + r^2 \dot{\theta}^2 + r^2 \dot{\varphi}^2 \sin^2 \theta$;
b) $\dot{r}^2 + \dot{\theta}^2 + \dot{\varphi}^2$;
c) $\dot{r}^2 + r^2 \dot{\theta}^2 + r^2 \dot{\varphi}^2 \sin^2 \theta$;
d) $\dot{r}^2 + \dot{\theta}^2 + \dot{\varphi}^2$.

2. Дано: $v_x(t) = 2t,\ x(0) = 1$. Найдите $x(t)$:
a) $t^2 + 1$;
b) $2t^2 + 1$;
c) $t^2$;
d) $2t$.

3. В цилиндрической системе координат локальный базисный вектор $\vec{e}_z$ равен:
a) $(\cos \varphi,\ \sin \varphi,\ 0)$;
b) $(-\sin \varphi,\ \cos \varphi,\ 0)$;
c) $(0,\ 0,\ 1)$;
d) $(0,\ 0,\ 0)$.

4. Частица движется в сферических координатах:
$ r(t) = 3t,\quad \theta(t) = \frac{\pi t}{3},\quad \varphi(t) = 2t.$
При $t = 1.5$ с $\sin(\pi/2) = 1$. Найдите модуль скорости $|v|$:
a) $\sqrt{9 + 20.25\pi^2 + 81}\ \text{м/с}$;
b) $\sqrt{9 + 2.25\pi^2 + 324}\ \text{м/с}$;
c) $\sqrt{9 + 20.25\pi^2 + 324}\ \text{м/с}$;
d) $\sqrt{9 + 2.25\pi^2 + 324}\ \text{м/с}$.

5. Полное ускорение:
a) $\ddot{r} \vec{\tau} + \frac{v^2}{R} \vec{n}$;
b) $\ddot{v} \vec{\tau} + \frac{v^2}{R} \vec{n}$;
c) $\ddot{s} \vec{\tau} + \dot{s}^2 \vec{n}$;
d) $\dot{v} \vec{\tau} - \frac{v^2}{R} \vec{n}$.

6. Частица движется по кривой: $v(t) = 2t^2 + 1,\ R = 4$ м. Найдите нормальное ускорение $|a_n|$ при $t = 2$ с.
a) $9\ \text{м/с}^2$;
b) $20.25\ \text{м/с}^2$;
c) $81\ \text{м/с}^2$;
d) $36\ \text{м/с}^2$.
