# Проект "Звездочет" 

### Концепт удобной образовательной системы для ВУЗов, СУЗов и ШКОЛ

## Наша цель
 Сделать образовательную систему интуитивно понятной каждому и работающей со скоростью сравнимой с третьей космической.

## !!ПРОЕКТ В АКТИВНОЙ РАЗРАБОТКЕ!!

## Наши фичи
 + Доступ к вспомогательным материалам для учебы
 + Тест-система с защитой от списывания
 + Интеграция с другими платформами
 + Возможность создания журнала с гибкой кастомизацией
 + Понятная документация(будет позже здесь)
 + Простота настройки и управления доступом
 + И многое другое...

## Наш стек технологий
 + React (Frontend)
 + Spring (Backend)
 + MySQL (Database)
 + Nginx (Web-server)
 + Docker Compose
 + Kubernetes
